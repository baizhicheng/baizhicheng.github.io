2019-12-25

Blogging with Hexo + Github Page

Preparation

  1. Install NodeJS

    Check if NodeJS is installed. Enter the following command to view the version information.

    node -v

  2. Install Git

    Check if Git is installed, enter the following command to view the version information.

    git --version

  3. Sign up for a Github account

    The registration process will not be repeated, and a repository is created after registering the account, and the warehouse name is: GitHubusername.github.io

Configure SSH key

​ Follow the prompts to complete three carriage returns to generate an ssh key.
